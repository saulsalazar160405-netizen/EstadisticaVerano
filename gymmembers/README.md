# Acerca del conjunto de datos

Este conjunto de datos ofrece una visión detallada de las rutinas de ejercicio, los atributos físicos y las métricas de condición física de los miembros de un gimnasio. Contiene **973 registros**, incluyendo indicadores clave de rendimiento como la frecuencia cardíaca, las calorías quemadas y la duración de los entrenamientos. Además, incorpora datos demográficos y niveles de experiencia, lo que permite realizar análisis completos sobre patrones de entrenamiento, progresión física y tendencias relacionadas con la salud.

## Características principales

| Variable | Descripción |
|----------|-------------|
| **Edad** | Edad del miembro del gimnasio. |
| **Género** | Género del miembro (Masculino o Femenino). |
| **Peso (kg)** | Peso corporal del miembro en kilogramos. |
| **Altura (m)** | Estatura del miembro en metros. |
| **Max_BPM** | Frecuencia cardíaca máxima (latidos por minuto) durante la sesión de entrenamiento. |
| **Avg_BPM** | Frecuencia cardíaca promedio durante la sesión de entrenamiento. |
| **Resting_BPM** | Frecuencia cardíaca en reposo antes del entrenamiento. |
| **Session_Duration (horas)** | Duración de cada sesión de entrenamiento en horas. |
| **Calories_Burned** | Total de calorías quemadas durante la sesión. |
| **Workout_Type** | Tipo de entrenamiento realizado (Cardio, Fuerza, Yoga, HIIT, entre otros). |
| **Fat_Percentage** | Porcentaje de grasa corporal del miembro. |
| **Water_Intake (litros)** | Cantidad de agua consumida durante los entrenamientos. |
| **Workout_Frequency (días/semana)** | Número de sesiones de entrenamiento realizadas por semana. |
| **Experience_Level** | Nivel de experiencia del miembro, donde **1 = Principiante**, **2 = Intermedio** y **3 = Experto**. |
| **BMI (IMC)** | Índice de Masa Corporal, calculado a partir del peso y la altura. |

## Aplicaciones del conjunto de datos

Este conjunto de datos es ideal para:

- Realizar análisis exploratorios de datos (EDA).
- Estudiar los hábitos de ejercicio y el rendimiento físico.
- Modelar la progresión del estado físico mediante técnicas de aprendizaje automático.
- Analizar la relación entre variables demográficas, fisiológicas y de entrenamiento.
- Desarrollar modelos predictivos para estimar calorías quemadas, frecuencia cardíaca o nivel de experiencia.
- Identificar patrones de entrenamiento y segmentar perfiles de usuarios.
- Investigar cómo distintos factores influyen en la intensidad del ejercicio, la resistencia y la salud general.

Gracias a la diversidad de variables incluidas, este conjunto de datos constituye un recurso valioso para científicos de datos, investigadores del área de la salud y profesionales del fitness interesados en obtener información relevante sobre el rendimiento físico y el comportamiento de los usuarios en un gimnasio.
